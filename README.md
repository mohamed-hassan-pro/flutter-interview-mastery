# 🚀 Flutter Interview Mastery - إتقان مقابلات فلاتر

A comprehensive web application designed specifically for Flutter developers preparing for interviews in the **Egyptian and Arab markets**. Featuring **150+ meticulously crafted cards** covering everything from Dart basics to complex system design.

## 🌟 مُميزات المشروع (Features)

- 📚 **150+ كارد مقابلة** - تغطية شاملة لجميع جوانب فلاتر ودارت.
- 🌐 **دعم كامل للغتين** - واجهة وتطبيقات باللغتين العربية والإنجليزية.
- 🌙 **الوضع الليلي (Dark Mode)** - تصميم عصري ومريح للعين.
- 🔍 **بحث ذكي (Smart Search)** - الوصول الفوري للمعلومة باستخدام Fuse.js.
- 🎯 **متابعة التقدم (Progress Tracking)** - سجل كامل لما قمت بمذاكرته.
- 📊 **قياس الثقة (Confidence Level)** - تقييم مستواك في كل موضوع (1-5 نجوم).
- 📖 **وضع الدراسة (Study Mode)** - بيئة مركزة للمذاكرة المتتابعة.
- 🧠 **وضع الاختبار (Quiz Mode)** - اختبر معلوماتك بأسئلة عشوائية وتحديات.
- 🏢 **فلاتر الشركات (Company Filters)** - أسئلة مخصصة لشركات محددة (Swvl, MaxAB, Fawry, etc.).

## 🛠 التكنولوجيا المستخدمة (Tech Stack)

*   **Logic:** React + TypeScript
*   **Styling:** Tailwind CSS + Framer Motion
*   **State Management:** Zustand (with Persistence)
*   **Architecture:** Clean & Modular Design

## 📦 البدء في التشغيل (Installation)

```bash
# انسخ المستودع
git clone https://github.com/mohamed-hassan-pro/flutter-interview-mastery.git

# انتقل للمجلد
cd flutter-interview-mastery

# التحميل
npm install

# التشغيل
npm run dev
```
